## Sample NODEJS backend project

**required to install**

- [NodeJS](https://nodejs.org/en/download/)
- [Postman](https://www.postman.com/downloads/)

**installation**

```
npm i
```

Run

```
node index.js
```

**How to create nodejs project**

```
npm init
```
